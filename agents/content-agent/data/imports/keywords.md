# Konu Listesi

Bu dosya Content Agent'ın TOPIC_RESEARCH skill'i tarafından doldurulur.
İnsan da buraya doğrudan konu ekleyebilir.

## Format

```
## [Numara]. [Başlık]
- Anahtar kelime: ...
- Segment: endüstriyel / medikal / mimari / reklam
- GEO potansiyeli: yüksek / orta / düşük
- Öncelik: 1–5
- Durum: bekliyor / taslak hazır / onaylandı / yayınlandı
```
